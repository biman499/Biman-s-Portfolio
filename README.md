A responsive and modern personal portfolio website showcasing my skills, services, and experience as a Web Developer, UI/UX Designer, and Photographer. 

## 🌟 Features

- **Responsive Design**: Fully responsive layout that looks great on desktops, tablets, and mobile devices.
- **Smooth Scrolling**: Easy navigation with smooth scroll behavior between sections.
- **Interactive UI**: Hover effects, responsive grids, and clean typography.
- **Working Contact Form**: Integrated with [EmailJS](https://www.emailjs.com/) to receive messages directly to my inbox without a backend server.
- **Sections Included**:
  - **Home**: Brief introduction and quick links.
  - **About**: Details about my background and roles.
  - **Services**: What I offer (Web Design, UI/UX, Photography).
  - **Skills**: A visual grid of my current toolkit and technologies I am learning.
  - **Contact**: A functional form for clients and collaborators to reach out.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and layout.
- **CSS3**: Custom styling, Flexbox/Grid layouts, and CSS variables.
- **JavaScript (Vanilla)**: DOM manipulation, smooth scrolling, and form handling.
- **EmailJS**: For handling the contact form submissions.
- **FontAwesome & Devicon**: For high-quality, scalable icons.

## 🚀 How to Run Locally

This project requires no build tools or server setup. To view it locally:

1. Clone or download the repository.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).
